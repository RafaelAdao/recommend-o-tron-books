# recommend-o-tron-books

The purpose of this project is to recommend books based on your reader-tracker (like skoob, goodreads, thestorygraph, etc) apps data

Currently working on a 0.0.1 working version.

## Techs

- Remix
- Fly
- NodeJS (express)
- openai

### Running

```sh
make setup
make run
make fake-server
```
